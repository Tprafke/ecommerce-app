# React E-commerce

### [Live Site](https://tprafke-react-ecommerce.herokuapp.com/)

## Description

Example E-commerce website built with React and .Net. Users can browse the product catalog, add items to their cart, and place orders. Admin users can manage the product inventory by adding, updating, or deleting products from the store.

## Features

<ul>
<li> Front end - React with TypeScript </li>
<li> Back end - .Net 6.0 API with C# </li>
<li> Database - PostgreSQL </li>
<li> Application published via Heroku </li>
<li> Authentication using ASP.NET Identity for login and registration </li>
<li> Client side inventory management for admin users </li>
<li> Checkout and ordering system </li>
<li> Stripe integration for accepting payments </li>
<li> Cloudinary integration for photo upload </li>
<li> Paging, sorting, searching, and filtering the product catalog </li>
<li> Material Design UI </li>
</ul>
